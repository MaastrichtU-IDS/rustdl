## Lessons from this session (Meta on the meta-lesson)

The patterns shipped or reverted this period add to the meta-lesson:

* **Measurement-gated multi-pass plans work.** Phase 2d alone gave
  zero corpus recovery (T4 gate passed but expected); shipping it
  unlocked Phase 2c-redux which then delivered the full 17-pair GALEN
  recovery. Without the intermediate gate the wall cost would have
  been impossible to attribute cleanly.
* **`rustdl explain` is the cheapest verification.** When `rustdl
  explain` and a corpus-diff disagree, prefer explain's signal — same
  code path with simpler scaffolding. Saved a 30-min re-measure on
  Anonymous-349.
* **Workload-dependent optimizations are §16-shape risky.** Per-edge
  HashMap probes (3e) and per-classify index construction (recon for
  3f Stage 2) both hit GALEN's edge-heavy / rule-thin profile and
  regress despite SIO flame wins. Default to workload-neutral fixes.
* **The advisor's "stop" call has been right multiple times this
  session** — Phase 3f recon, Anonymous-349, the trust_sat auto-gate
  trade-off. When the cost-asymmetry says "spend another 30 min to
  maybe shave 1 pair", the right answer is usually stop.
* **CPU flame can lie by omission** (Phase 5 T1). A regression
  concentrated in a single-threaded section will be invisible against
  multi-core sampling. When wall-time and flame disagree, instrument
  with `Instant::now()` deltas directly — see Phase 5 T2/T3b docs for
  the pattern.
* **Diagnostic-mode-first beats default-change-first for soundness
  contracts** (Phase 4b). Shipping the fragment classification as a
  banner + `ClassificationStats` field surfaced the contract without
  wall risk; the auto-gate version (Phase 4a) can layer on top if
  ever wanted. Smaller increment, larger immediate value.
* **A regression can be inherent**, not a bug. Phase 5 T3b traced
  the +6.5 % GALEN wall to the tier-walk cost of the 17 newly
  recovered closure pairs — proportional to the completeness gain.
  Calling it "a regression to fix" was the wrong framing; the right
  framing is "the price the completeness work paid."
